# Sales Performance Dashboard

## Project Overview
This project analyzes business sales performance using Tableau and Excel. The dashboard provides insights into regional sales trends, monthly revenue performance, and product category revenue.

## Tools Used
- Tableau Public
- Microsoft Excel
- Data Visualization
- Business Analytics

## Key Features
- Revenue by Region analysis
- Monthly revenue trend visualization
- Product category revenue comparison
- Interactive dashboard filtering
